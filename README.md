# 🚗 EV Infrastructure Intelligence System

Advanced Machine Learning-powered dashboard for EV adoption forecasting and infrastructure planning in Maharashtra, India.

## 🎯 Features

- **📊 Interactive Dashboard** - Real-time KPIs and visualizations
- **🤖 ML Predictions** - Linear regression forecasting for EV growth
- **🗺️ Geographic Mapping** - RTO-wise infrastructure demand visualization
- **📈 Multi-page Analytics** - EV adoption, infrastructure readiness, city comparison
- **🔌 Gap Analysis** - Charger infrastructure requirement calculations
- **📊 Rich Visualizations** - Plotly charts, pie charts, gauges, and heatmaps
